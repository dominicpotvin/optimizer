Calcul de percage - Profile en U

Repartition symetrique des trous a partir du centre du profile.

Methode de calcul

Repartition symetrique autour du centre du profile. Le nombre de trous N est choisi pour que la marge de bout (bord du profile au centre du trou extreme) reste dans l'intervalle [marge min, marge max]. Si N est impair, un trou est au centre exact ; si N est pair, les trous encadrent le centre a +/- (entraxe / 2). On retient le N valide qui place les trous extremes le plus pres possible de la marge max, en gardant l'equilibre. Si aucune configuration ne respecte les deux bornes, le calcul est declare impossible pour cette longueur.
